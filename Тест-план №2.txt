Проверка сервиса на сайте - https://randomuser.me/
В данном API можно реализовать гораздо больше тестов, но я решил, что для тестового задания этого достаточно
1)Проверка отправки запроса без параметров (автоматизирован)
2)Проверка отправки запроса с параметром (автоматизирован)
3)Проверка отправки 2х параметров (автоматизирован)
4)Проверка отправки параметра на   исключения полей  (автоматизирован)

Какие бы проверки я еше добавил:
5)Проверка отправки параметра с ошибкой(сейчас запрос выполняется если отправить не существующий параметр, а должна быть ошибка)
6) Проверка отработки запроса с параметром на пароли
7)Проверка отработки запроса с параметром на форматы
8)Проверка отработки запроса с старой версией API
9)Проверка отработки при запросе нескольких пользователей
